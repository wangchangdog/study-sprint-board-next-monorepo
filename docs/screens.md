# 画面一覧

## 画面遷移

```
/signin → /dashboard
                ├── /tasks
                │     ├── /tasks/new
                │     ├── /tasks/[taskId]
                │     └── /tasks/[taskId]/edit
                └── /settings (admin のみ)
```

## 各画面の仕様

### /signin — サインイン

- メールアドレスとパスワードによるログイン
- 開発用アカウント情報を画面下部に表示
- 認証成功時は `/dashboard` にリダイレクト

### /dashboard — ダッシュボード

- ステータス別タスク件数（TODO / IN_PROGRESS / IN_REVIEW / DONE）
- 自分の担当タスク数
- 締切が近いタスク一覧（7日以内）
- 最近更新されたタスク一覧

### /tasks — タスク一覧

- 全タスクをテーブル形式で表示
- 列: タイトル、ステータス、優先度、担当者、締切日
- コメント数をタイトル横に表示
- 「新規作成」ボタン → /tasks/new

### /tasks/new — タスク作成

- フォーム: タイトル（必須）、説明、ステータス、優先度、締切日、担当者、ラベル
- Zod バリデーション
- 作成成功後は /tasks にリダイレクト

### /tasks/[taskId] — タスク詳細

- タスク情報の詳細表示
- コメント一覧
- サイドバー: ステータス、優先度、担当者、作成者、締切、ラベル、作成日
- 「編集」ボタン → /tasks/[taskId]/edit

### /tasks/[taskId]/edit — タスク編集

- タスク作成と同じフォーム（既存値をプリフィル）
- PATCH メソッドで更新

### /settings — 設定（admin のみ）

- ユーザー一覧（名前、メールアドレス、権限）
- admin 以外がアクセスすると /dashboard にリダイレクト
- 将来拡張: ラベル管理、システム設定
